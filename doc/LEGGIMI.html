Note in ordine sparso

Questo programma consente di scaricare da Internet le puntate della trasmissione radiofonica "La storia in giallo" salvandole in un file MP3, molto comodo per l'uso sia sul computer sia su lettori musicali esterni. I file MP3 vengono salvati sulla scrivania (desktop), in una directory chiamata La storia in giallo.

Alcuni avvisi:

  1. lo scaricamento richiede più o meno la stessa durata della puntata, in media tra i 40 ed i 50 minuti, abbiate pazienza;
  2. durante lo scaricamento viene creato un file temporaneo nella directory menzionata sopra che può raggiungere dimensioni di poco inferiori a 500Mb, assicuratevi di avere abbastanza spazio su disco! In ogni caso tale file viene cancellato una volta creato l'MP3.
  3. i file MP3 hanno dimensioni nell'ordine dei 20Mb.

Per domande o qualsiasi altro motivo, scrivetemi pure all'indirizzo email villa.lobos chiocciola tiscali.it.

Istruzioni specifiche per Windows

Se usate Windows Vista, 7, o successivi, potete tranquillamente installare il programma ed usarlo a vostro piacimento.

Se avete Windows XP, prima di installare il programma dovete scaricare ed installare il Framework .NET fornito da Microsoft, disponibile qui.

In generale consiglio di usare questo programma su un qualsiasi sistema che supporti il Framework .NET versione 3 o superiore.

Istruzioni specifiche per Linux

Quale che sia la distribuzione Linux che state usando, assicuratevi di avere installato i pacchetti mplayer, lame e mono (versione 2.4 o superiore). Se è tutto a posto non dovete far altro che eseguire il file "La storia in giallo.exe" facendo doppio click sullo stesso (se siete fortunati) o lanciando da terminale il comando

mono 'La storia in giallo.exe'.

In ogni caso, se usate Linux mi aspetto che siate in grado di far funzionare questo programma senza troppi aiuti. Al momento non ci sono pacchetti per nessuna distribuzione, se volete aiutare in tal senso, fatevi pure avanti!
